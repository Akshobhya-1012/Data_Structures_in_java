# Data Structures in Java
 
This repository contains **from-scratch implementations** of core data structures in **Java**, with a focus on:
 
- Understanding **internal working**, not just using library classes
- Clean, readable code suitable for **interviews**
- Documented **time & space complexity** for each operation
 
These implementations help me revise quickly before interviews and also demonstrate my understanding as a **Java backend developer (3+ years)**.
